# O Rummikub
	Trabalho 4 de ICC I (professor Adenilso da Silva Simão)
	O programa foi compilado com o GNU GCC Compiler, então a forma ideal de compilá-lo é com esse
compilador. Há, no repositório, um executável (versão compilada), então pode-se tentar executar o pro-
grama por lá, sem que haja a necessidade de compilação.
	Entradas-exemplo e um tutorial estão presentes em um pdf em anexo. Em anexo também estão 3 ar-
quivos de código-fonte (.c) e 1 cabeçário (.h). O versionamento foi feito através do GitHub.
